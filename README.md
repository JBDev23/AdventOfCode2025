# Advent of Code 2025

Repositorio con mis soluciones para los retos de programación diarios del [Advent of Code 2025](https://adventofcode.com/2025).

Como estudiante de Ingeniería del Software y desarrollador, utilizo este evento anual para poner a prueba mi lógica, optimizar estructuras de datos y mejorar mis habilidades de resolución de problemas algorítmicos.

---

## Tecnologías Utilizadas

* **Python**: Lenguaje principal para la resolución de los retos gracias a su versatilidad y rapidez para el scripting.

---

## Estructura del Repositorio

El código está organizado de forma secuencial por días. Cada directorio contiene los datos de entrada personalizados (`input.txt`) y el script con la solución.

```text
advent-of-code-2025/
├── Day 1/
│   ├── Input.txt
│   └── Secret_Entrance_P1.py
│   └── Secret_Entrance_P2.py
├── Day 2/
│   ├── Input.txt
│   └── Gift_Shop_P1.py
│   └── Gift_Shop_P2.py
└── ...
```
